---



\## 🎯 Opis problemu



Problem plecakowy polega na wyborze optymalnego zestawu przedmiotów w taki sposób, aby:



\- nie przekroczyć określonego ograniczenia (np. budżetu, pojemności magazynu, limitu promocji),

\- zmaksymalizować wartość wybranych elementów (np. zysk, atrakcyjność oferty, sprzedaż).



W naszym przypadku:



\- 🎽 Przedmioty → ubrania dostępne w sklepie internetowym  

\- 💰 Ograniczenie → np. budżet klienta lub limit kosztów  

\- 📈 Wartość → np. zysk, popularność produktu lub marża  



---



\## ⚙️ Zastosowane podejście



W projekcie wykorzystano algorytmy sztucznej inteligencji do:



\- modelowania problemu,

\- optymalizacji wyboru produktów,

\- wyznaczenia najlepszego możliwego zestawu ubrań przy zadanych ograniczeniach.
